# 📊 Sales Dashboard using Microsoft Excel

## 🔍 Project Overview
This project showcases an interactive Sales Dashboard developed entirely using Microsoft Excel. The dashboard provides a comprehensive analysis of sales performance by visualizing key business metrics such as total revenue, profit, sales growth, regional performance, and product category analysis.

The objective of this project is to transform raw sales data into meaningful insights that support data-driven business decisions.

## 🛠 Tools & Techniques Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Data Cleaning & Formatting
- Conditional Formatting
- Data Validation
- KPI Indicators
- Excel Formulas (SUM, IF, VLOOKUP/XLOOKUP, etc.)

## 📈 Key Dashboard Features
- Total Revenue and Profit Overview
- Region-wise and Category-wise Sales Analysis
- Monthly and Yearly Sales Trends
- Top-Selling Products Identification
- Interactive Filters and Slicers
- Dynamic Charts and KPI Cards

## 📊 Key Insights Generated
- Identified top-performing regions and products
- Analyzed seasonal sales fluctuations
- Evaluated profit margins across categories
- Highlighted areas requiring performance improvement

## 🎯 Business Value
This dashboard enables stakeholders to monitor performance effectively, identify growth opportunities, and make strategic decisions based on structured data analysis.
